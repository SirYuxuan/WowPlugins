# 雨轩工具箱功能清单

本文档用于整理当前插件的功能分组、模块作用与常用指令，便于后续维护与继续扩展。

## 新模块划分（重构后）

### 界面美化


- 聊天美化
  - 优化聊天框显示、字体、透明度与频道表现
- 游戏条
  - 左侧按钮、右侧按钮、炉石快捷交互
  - 支持背景、悬停、大小、间距、时间样式配置
- 系统调节
  - 战斗文字缩放、鼠标提示优化、目标标记显示等界面微调


### 战斗辅助

- 快捷频道
  - 快速切换说话、喊话、队伍、团队、公会、世界等频道
  - 支持自定义按钮、顺序、颜色、字体、拖动位置
- 施法条
  - 玩家、目标、焦点、GCD 施法条
  - 支持材质、尺寸、位置、透明度、颜色、字体设置
- 距离监控
  - 显示与目标的大致距离区间
  - 支持刷新频率、背景、边框、字体设置
- 团队标记
  - 快捷标记、清除标记、就位确认、倒计时
  - 支持横竖排、图标大小、背景与边框设置


### 信息显示

- 专精/天赋信息条
  - 显示专精、天赋方案、耐久等信息
- 属性显示
  - 显示装等、主属性、暴击、急速、精通、全能等
  - 支持颜色、背景、进度条、字体设置
- 货币展示
  - 自选货币显示
  - 支持横向/纵向、图标/文字/混合显示、顺序管理
- 升级提示
  - 显示经验效率、剩余经验、预计升级时间
- 性能监控

  - 显示延迟、帧数、内存等性能信息

### 社交与聊天

- 快捷频道（已归类至战斗辅助，但功能上属于社交）
- 任务助手

  - 任务相关快捷操作与通报能力
  - 支持按钮布局、字体与文字颜色设置

### 副本工具

- 副本难度助手
  - 快速切换地下城/团队难度
  - 支持重置副本、传进/出副本、一键退出
  - 支持按钮选中/未选中颜色自定义

- 地下堡快速离开
  - 地下堡内显示快速离开图标
  - 支持测试显示、图标预设、自定义图标、拖动位置

### 地图与探索

- 地图标记
  - 地图功能点、NPC、服务点、传送门等标记
  - 支持自定义坐标、导入导出、快速导航
  - 支持坐标显示与地图下方快捷输入

- 事件追踪器
  - 显示版本事件、周常与循环活动状态
  - 当前按版本分组：至暗之夜、地心之战
  - 支持地图下方显示、聊天通知、中央提示、样式设置

### 实用工具

- 收益计时器
  - 统计一段时间内的金币和经验变化
- 配置档

  - 通过 AceDB 管理插件配置档
- 关于
  - 展示版本、主要模块、常用命令与使用提示

## 常用命令

- `/yx`：打开设置
- `/yx lock`：锁定施法条
- `/yx unlock`：解锁施法条
- `/yx diff`：显示或隐藏副本难度助手
- `/yx diff reset`：重置当前副本
- `/yx diff leave`：执行一键退出
- `/c`：显示或隐藏副本难度助手
- `/timer`：打开收益计时器
- `/yxpin`：快速添加当前位置坐标
