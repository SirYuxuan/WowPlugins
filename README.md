# 雨轩工具箱（YuXuanToolbox）

**雨轩工具箱（YuXuanToolbox）** 是一个轻量、模块化的《魔兽世界》多功能插件。  
它将聊天工具、属性显示、货币统计、施法条以及地图标记等常用功能整合在一起，让你用更少的插件完成更多事情。

---

## 主要功能

### 快捷频道条

- 一键切换 **说话 / 喊话 / 队伍 / 团队 / 公会 / 世界** 等频道
- 支持：
  - 自定义按钮
  - 按钮排序
  - 颜色设置
  - 字体调整

---

### 属性显示面板

- 显示角色重要信息：
  - 装等
  - 主属性
  - 暴击
  - 急速
  - 精通
  - 全能
- 支持：
  - 单项显示开关
  - 颜色自定义
  - 字体与样式设置

---

### 货币展示

- 按分类选择需要显示的货币
- 支持多种布局方式：
  - 横向排列
  - 纵向排列
  - 图标模式 / 文本模式
- 支持自定义顺序

---

### 自定义施法条

支持多种施法条：

- 玩家施法条
- 目标施法条
- 焦点施法条
- GCD 施法条

可自定义：

- 尺寸
- 位置
- 透明度
- 颜色
- 字体
- 材质

可选功能：

- 隐藏暴雪默认施法条

---

### 全地图 NPC / 功能点标记（Map Guide）

在世界地图上显示各种有用标记。

**功能：**

- 多类型标记（文本 / 图标）
- 鼠标提示信息
- 支持按专业过滤

**自定义坐标功能：**

- 快速添加坐标
- 坐标列表管理
- 导入 / 导出坐标

**导航功能：**

- 点击地图标记即可设置导航点（Waypoint）

**坐标显示：**

- 在世界地图左下角显示：
  - 当前玩家坐标
  - 鼠标所在坐标

---

## 指令

| 指令                        | 功能                  |
| --------------------------- | --------------------- |
| <code>/yx</code>            | 打开插件设置          |
| <code>/yx lock</code>       | 锁定施法条            |
| <code>/yx unlock</code>     | 解锁施法条（可移动）  |
| <code>/yx diff</code>       | 显示/隐藏副本难度助手 |
| <code>/yx diff reset</code> | 重置当前副本          |
| <code>/yx diff leave</code> | 执行一键退出          |
| <code>/c</code>             | 显示/隐藏副本难度助手 |
| <code>/timer</code>         | 打开收益计时器        |
| <code>/yxpin</code>         | 快速添加当前位置坐标  |
